# bash-scripting
a repo containing resources on bash scripting

Notes:
- `#!/bin/bash` is called a shebang; indicates that it is a bash script
- anything after # is a comment


Check permissions:
- view permissions: `ls -l`
- Change permissions: `chmod 755 myscript.sh`
- run it `./myscript.sh`
